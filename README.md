# Django CRM Project

## Descripción del Proyecto

Este proyecto consiste en un **Sistema de Gestión de Relaciones con Clientes (CRM)** y **Control de Agendamientos**, desarrollado en **Python** utilizando el framework **Django**.

La plataforma está diseñada para administrar de manera centralizada la información operativa de una empresa orientada a servicios, estructurándose en torno a dos módulos principales.

---

## Características

### Gestión de Directorio de Clientes

Permite almacenar y consultar perfiles completos de clientes.

Información registrada:

- Nombre
- Correo electrónico
- Teléfono
- Dirección
- Ciudad
- Estado
- Código postal

Esto facilita el seguimiento de clientes y la comunicación directa con ellos.

---

### Control de Agendamientos y Facturación

El sistema incorpora un módulo para la administración de citas y servicios.

Cada agendamiento almacena:

- Responsable del servicio
- Concepto del servicio
- Fecha y hora programada
- Precio total

Este módulo permite llevar un control organizado de los servicios prestados y de los ingresos proyectados.

---

## Objetivo

El proyecto proporciona una solución **backend escalable** para:

- Centralizar la información de clientes.
- Optimizar los procesos administrativos.
- Gestionar citas de forma estructurada.
- Llevar un control básico de facturación.

---

# Diseño de la Interfaz

Además del backend, el proyecto incorpora una interfaz moderna con una identidad visual personalizada.

## Bootstrap 5

La aplicación utiliza **Bootstrap 5** como framework principal para ofrecer:

- Diseño responsive (Mobile First)
- Sistema de grillas
- Componentes reutilizables
- Consistencia visual

---

## Estética CyberSigilism

La interfaz implementa un estilo visual inspirado en el **CyberSigilism**, utilizando tipografías de **Google Fonts** como:

- Orbitron
- Rajdhani
- Share Tech Mono

Estas fuentes aportan una apariencia futurista y tecnológica.

---

## Efectos Visuales

La experiencia de usuario se complementa con efectos desarrollados mediante **CSS** y **JavaScript**, entre ellos:

- Fondo de partículas interactivas
- Scanlines animadas
- Personalización avanzada del tema
- Animaciones sutiles de interfaz

El objetivo es ofrecer una experiencia visual moderna y diferenciada de los paneles administrativos tradicionales.
